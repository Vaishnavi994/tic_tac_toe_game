# Tic Tac Toe Game 🎮

## 📌 Project Overview
Tic Tac Toe is an interactive two-player game developed using programming concepts such as game logic, user input handling, and condition checking. The application allows players to compete against each other by placing their symbols on a 3×3 grid and determines the winner based on predefined game rules.

## 🎯 Objective
The objective of this project is to build a simple and interactive Tic Tac Toe game that implements game mechanics, player turns, win detection, and draw conditions.

## ✨ Features
- Two-player gameplay.
- Interactive 3×3 game board.
- Automatic detection of winning combinations.
- Detects draw situations.
- Displays game results clearly.
- Simple and user-friendly interface.

## 🕹️ How It Works

1. The game starts with an empty 3×3 board.
2. Players take turns placing **X** and **O** symbols.
3. After every move, the system checks for:
   - Winning combinations.
   - Draw condition.
4. The game displays the final result.
